## Core Features

- User Registration with National ID verification, photo, and ID scan
- Gig posting with skill categories, urgency levels, and budget
- Gig acceptance with instant chat between worker and client
- Live GPS location sharing stored in admin logs
- Emergency button with Police, Fire, Ambulance hotlines
- Strike and penalty system (6 strikes = disqualification)
- Password reset with identity verification
- Admin panel for fraud prevention and emergency response
- Bill of Rights integration (Constitution of Kenya, 2010)
- Kabambe version for non-smartphone users
- Rotating Kenyan landscape backgrounds
